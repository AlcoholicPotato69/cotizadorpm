# 03 - Operación 100% Offline (sin Internet)

Este repo ya incluye `client/supabase.js` (librería local).  
Aún así, si tu PC/servidor **no tendrá Internet**, revisa:

- Tailwind CDN: `https://cdn.tailwindcss.com`
- FontAwesome CDN: `https://cdnjs.cloudflare.com/...`

Para modo offline real, hay que “vendorizar” (descargar) esos recursos y apuntar a archivos locales.
